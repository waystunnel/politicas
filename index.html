1. Introdução

A MTUs VPN é um aplicativo desenvolvido para fornecer uma conexão VPN segura e privada para dispositivos Android. Esta Política de Privacidade descreve como tratamos as informações dos nossos usuários.

A nossa prioridade é a transparência. Por isso, explicamos aqui todos os detalhes sobre o tratamento de dados no nosso aplicativo.

Compromisso com a privacidade: A MTUs VPN não coleta, armazena ou compartilha dados pessoais dos usuários. A sua privacidade é respeitada integralmente.

2. Coleta de dados

A MTUs VPN não coleta dados pessoais dos seus usuários. Não solicitamos, armazenamos ou processamos informações pessoais identificáveis.

Não coletamos:

  • Nomes, e-mails ou números de telefone.
  • Localização GPS ou endereço IP.
  • Contatos, fotos, arquivos ou qualquer conteúdo do dispositivo.
  • Informações do dispositivo, como modelo ou versão do sistema operacional.
  • Dados de navegação, histórico de sites ou conteúdo trafegado.
  • Logs de erro ou dados de desempenho.
  • Cookies ou identificadores de publicidade.
  • Qualquer outro dado pessoal identificável.

3. Funcionamento da VPN

A MTUs VPN estabelece uma conexão segura entre o dispositivo do usuário e os nossos servidores. Este processo é feito sem qualquer coleta ou armazenamento de dados pessoais.

A permissão VpnService, exigida pelo Android, é utilizada exclusivamente para criar o túnel de comunicação criptografado. Em nenhuma circunstância utilizamos essa permissão para acessar, modificar ou coletar dados dos usuários.

4. Serviços de terceiros

A MTUs VPN integra serviços de terceiros para garantir o funcionamento do aplicativo e melhorar a experiência do usuário. Estes serviços são:

  • Firebase (Google): Utilizado para notificações push, autenticação de usuários e monitoramento de desempenho do aplicativo. Todos os dados processados pelo Firebase são anonimizados e não permitem identificação pessoal.
  • Google Analytics (Firebase Analytics): Coleta dados agregados e anônimos sobre o uso do aplicativo, como telas acessadas e duração das sessões. Essas informações nos ajudam a entender como os usuários interagem com o aplicativo para melhorá-lo continuamente. Não coletamos IDs de publicidade ou dados pessoais através deste serviço.
  • Google AdMob: Exibe anúncios dentro do aplicativo para manter o serviço gratuito. O AdMob pode coletar dados de publicidade anonimizados para exibir anúncios mais relevantes. Não compartilhamos nenhum dado pessoal com o AdMob. Os usuários podem gerenciar suas preferências de anúncios diretamente nas configurações do dispositivo.

Reforçamos que esses serviços são utilizados apenas para fins operacionais e estatísticos. Nenhum dado pessoal é coletado, vendido, compartilhado ou transferido a terceiros para finalidades não relacionadas ao funcionamento do aplicativo.

5. Segurança

Embora não coletemos dados pessoais, implementamos medidas técnicas para garantir a segurança da conexão VPN, protegendo o tráfego de dados dos usuários contra interceptações e acessos não autorizados.

6. Compartilhamento com terceiros

A MTUs VPN não compartilha dados pessoais com terceiros, uma vez que não coletamos informações pessoais dos usuários.

7. Conformidade com o GDPR e a LGPD Em conformidade

A MTUs VPN está em conformidade com o Regulamento Geral de Proteção de Dados (GDPR) da União Europeia e com a Lei Geral de Proteção de Dados (LGPD) do Brasil. Como não coletamos dados pessoais, não há informações a serem gerenciadas, acessadas ou excluídas.

Caso tenha qualquer dúvida sobre privacidade, entre em contato conosco através do e-mail informado nesta política.

8. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças no aplicativo ou melhorias no serviço. Recomendamos que os usuários revisem esta página regularmente para se manterem informados.
